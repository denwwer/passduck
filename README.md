# PassDuck

A lightweight open-source Chrome and Safari extension for instantly generating secure, random passwords.
Customize length, character sets, and easily copy with one click — all processed locally for privacy.

## Features

- ✅ **Lowercase** letters (a-z)
- ✅ **Uppercase** letters (A-Z)
- ✅ **Numbers** (0-9)
- ✅ **Custom characters** (configurable)
- ✅ **Strength entropy** meter
- ✅ **Passphrase** generator ([fun fact](https://xkcd.com/936/))
- ✅ **Colorized** password characters
- ✅ **"Blur"** to keep _shoulder-surf_ protection
- ✅ **Persisted states** in settings `chrome.storage.sync`
- 📋 **One-click copy** to clipboard
- 🌙 **Dark theme** support (automatic based on system preference)
- 🔒 **Cryptographically secure** random generation and Fisher-Yates shuffle

## Browser Compatibility

- Safari 15+ (custom build)
- Chrome 88+
- Edge 88+
- Brave
- Any Chromium-based browser with Manifest V3 support

## Support

For issues or feature requests, please modify the code as needed. This is a standalone extension with no external dependencies.
